# 📊 Análise Estratégica de Vendas no E-commerce Brasileiro

## 📌 Sobre o Projeto

Projeto desenvolvido utilizando Python para análise e visualização de dados de um marketplace brasileiro.

A análise foi realizada com foco em:
- faturamento
- comportamento dos consumidores
- categorias de produtos
- avaliações
- formas de pagamento
- correlação entre variáveis

---

## 🚀 Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset

Brazilian E-Commerce Public Dataset by Olist:

https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

---

## 📈 Principais Análises

- Evolução da receita ao longo do tempo
- Top categorias por faturamento
- Distribuição das avaliações
- Formas de pagamento
- Matriz de correlação
- Relação entre preço e valor pago

---

## 📌 Principais Insights

- Crescimento consistente do faturamento
- Forte presença das categorias de casa, beleza e tecnologia
- Predominância de avaliações positivas
- Cartão de crédito como principal forma de pagamento
- Forte correlação entre preço e valor pago
